# Online Reservation System

The Online Reservation System is a Java-based application that allows users to make and cancel reservations. This system is a console-based prototype designed to demonstrate the basic functionality of a reservation system.

## Features

- **Make a Reservation:** Users can make a reservation by entering their name.
- **Cancel a Reservation:** Users can cancel a reservation by entering their name.
- **View Reservations:** Users can view a list of current reservations.


## Usage

1. Choose an option from the menu by entering the corresponding number.
2. Follow the prompts to make a reservation, cancel a reservation, or view existing reservations.
3. Exit the application when finished.
